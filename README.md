# KPOP Video

KPOP Video Player application.

## Prerequisites

- Node.js (v18+)

## Setup & Run

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Structure

- `src/` : Application source code.
  - `scripts/`: JS logic divided by domains.
  - `styles/`: Modular CSS setup.
  - `assets/`: Images, Fonts, Tutorial resources.

## Development Tools

- **Vite**: Fast dev server and bundler.
- **ESLint & Prettier**: Code linting and formatting.
- **JSDoc/TypeScript checking**: Provides type safety in JavaScript files (`checkJs`).
